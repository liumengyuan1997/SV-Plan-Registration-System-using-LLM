Database Design Project (Group 4): SV Plan Registration System using LLM

The SV Plan Registration System is designed to facilitate task management and event announcements for the SV campus community. This comprehensive system is managed by Khoury College, and supports both web-based and mobile-based platforms, enabling users to efficiently track their daily tasks through photo uploads, file attachments, or voice memos. Additionally, the system keeps users informed about crucial campus events, ensuring they remain engaged and up-to-date.
